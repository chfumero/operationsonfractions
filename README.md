# Operations on fractions

## Requirements
```shell script
python3
pip3
```

## How to use

```shell script
$ git pull https://github.com/chfumero/operationsonfractions.git
$ cd operationsonfractions
$ sh install.sh
$ operationsonfractions "5/3"
1_2/3
```
